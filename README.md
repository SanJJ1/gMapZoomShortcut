# Google Maps Keyboard Nav

**Chrome Web Store:** https://chromewebstore.google.com/detail/google-maps-keyboard-nav/ehfkgkjppaamcbigkancpjcijnkhlajp

## Shortcuts

| Key(s) | Action |
|--------|--------|
| `+` / `=` / `i` | Zoom in |
| `-` / `_` / `o` | Zoom out |
| `h` | Pan left |
| `l` | Pan right |
| `k` | Pan up |
| `j` | Pan down |

Hold two pan keys for diagonal movement. Tap for precise nudges, hold for fast smooth scrolling with acceleration. All shortcuts disable when a text field is focused.

## Chrome Web Store Fields

**Title from package** (`manifest.json` → `name`)

    Google Maps Keyboard Nav

**Summary from package** (`manifest.json` → `description`)

    Fixes Google Maps keyboard shortcuts. Reliable +/- zoom and HJKL vim-style panning with smooth acceleration.

**Description**

```
Google Maps' native keyboard shortcuts are broken — the +/- keys work intermittently and arrow key panning is painfully slow.

This extension replaces them with responsive, reliable controls:

── ZOOM ──
• +  /  =  /  I  →  zoom in
• -  /  _  /  O  →  zoom out

── PAN ──
• H / J / K / L  (vim-style)
• Tap for precise nudges, hold for fast smooth scrolling
• Supports diagonal movement by holding two keys at once

All shortcuts automatically disable when a text field is focused (search bar, etc).

A small personal tool — no tracking, no permissions, no nonsense.

https://github.com/SanJJ1/gMapZoomShortcut
```
